# Hydrogen-Natural Gas Blending Analysis

Code for manuscript "Can Renewable Hydrogen Blending Support a Nationwide Pathway in China"

## Description

Optimization model for electrolyzer capacity using curtailed renewable electricity data, including economic and lifecycle emission analysis for hydrogen-natural gas blending in China.

## Electrolyzer Types

- PEM (Proton Exchange Membrane)
- Alkaline

Parameters can be adjusted to switch between electrolyzer types.

## Functionality

- Capacity optimization
- Cost calculation
- Emission assessment
